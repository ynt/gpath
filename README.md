# gpath
Easy to use GOPATh

# Start use
```
$ cd ~/Go/src/github.com/ynt/gpatch
$ path/to/gpath
// output ~/Go/
```

alias you want use GOPATH
```
alias go='export GOPATH=$(/usr/local/bin/gpath); go'

alias dep='export GOPATH=$(/usr/local/bin/gpath); dep'
alias glide='export GOPATH=$(/usr/local/bin/gpath); glide'
```
